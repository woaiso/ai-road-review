- 对于个人来讲，在当前这个时代对于软件从业者是最好的时候，我们以前需要花大量时间才能推出的想法，现在可能几个小时就实现了。
  - 现在焦虑的不是失业的问题，而是Token是否够用
- 对于团队来讲，如果在AI时代还是用旧的组织结构、旧的管理方法和流程去做事的话必然会失败
  - 钉钉CEO 无招，未来中层将不复存在
- 规则驱动 AI（早期 AI）
  - 如果 发烧 + 咳嗽 → 可能是感冒
  - 如果 A 且 B → 推导 C
  - 本质：人工写死的智能
- 机器学习（Machine Learning）
  - 不写规则，让机器“从数据中学规律”
  - 垃圾邮件过滤、信用评分、推荐系统
  - 本质：从数据中拟合函数
- 深度学习（Deep Learning）
  - 神经网络
  - 人脸识别
  - 语音识别
  - 本质：用神经网络自动提取特征
- 判别式 AI（Discriminative AI）
  - 输入 X → 预测 Y
  - 输入一张猫的图片，问这是不是猫
  - 本质：分类 / 回归
- 生成式 AI（Generative AI）← 现在这波
  - 不只是判断，而是“生成新内容”
  - 学的是“数据分布”，不是简单映射
